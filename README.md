# FocusDice

Минимальное Android-приложение на Kotlin.

## Что исправлено
- Добавлен `gradlew`
- Добавлен workflow для GitHub Actions
- Workflow использует `actions/upload-artifact@v4`
- `gradlew` умеет автоматически скачать `gradle-wrapper.jar`, если его нет в репозитории

## Как собрать APK на GitHub
1. Загрузи содержимое этой папки в корень репозитория.
2. Убедись, что файл `.github/workflows/android.yml` тоже загружен.
3. Сделай commit в ветку `main`.
4. Во вкладке **Actions** дождись успешной сборки.
5. Скачай артефакт `app-debug`.

## Если снова будет ошибка
Пришли лог из шага, который упал. Чаще всего это либо неполная загрузка файлов в репозиторий, либо проблема с путями.
